## 🎯 Project Overview

A **production-ready** Local RAG (Retrieval-Augmented Generation) system that enables intelligent questioning and answering from your documents. Built with modern LLM architecture, this system processes documents, stores them in a vector database, and provides accurate answers using state-of-the-art language models.

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| 📄 **Multi-Format Support** | Upload PDF, DOCX, and TXT files |
| 🔍 **Intelligent Retrieval** | FAISS vector search for semantic similarity |
| 🤖 **Advanced LLM** | Powered by Groq's Llama 3.3 70B model |
| 💾 **Persistent Storage** | No data loss on restart |
| 🚀 **High Performance** | Sub-second response times |
| 🎨 **Modern UI** | Clean, professional Streamlit interface |
| 🔒 **Local Deployment** | 100% private, no data leaves your machine |
